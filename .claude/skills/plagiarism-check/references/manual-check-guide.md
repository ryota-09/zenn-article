# 手動剽窃チェックガイド(CopyContentDetector)

AIエージェントによる自動チェックの最終確認として、公開直前に1回だけ手動で実施することを推奨する。

## 手順

1. https://ccd.cloud/ (CopyContentDetector) を開く
2. 記事本文を4,000字ずつに分割してコピー&ペーストする(無料枠は4,000字までの制限があるため)
3. 各チャンクでチェックを実行する
4. 「一致率」が50%を超える箇所があれば、該当箇所を目視で確認し、必要なら書き換える

## 位置づけ

これはAIエージェントの `plagiarism-checker` による検索ベースのチェックを置き換えるものではなく、公開直前の最終防波堤として運用する。個人のZenn運用では有料API(Copyscape等)を導入するコストに見合わないため、この無料ツールでの目視確認を採用している。

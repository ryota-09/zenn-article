## 設計思想

LLMによるパラフレーズはLLM同士の比較検出をすり抜けやすい。そのため**必ずWeb検索での裏取りを組み合わせる**。サブエージェントは証拠収集のみを行い、剽窃かどうかの判定はこのスキル(あなた自身)が行う。

## 手順

1. 参考URLを収集する。優先順位: 引数で渡されたURL → 記事内の「参考資料」セクション → 会話の文脈にあるURL
2. `plagiarism-checker` サブエージェントを起動し、記事パスと参考URLを渡す
3. 返ってきた証拠をもとに、あなた自身が3段階で判定する:
   - **レベルA(要修正)**: 20字以上の連続一致 / 見出し構成がほぼ同一 / コード例が出典なしで一致
   - **レベルB(書き換え推奨)**: 論理展開の順序が特定記事と酷似 / 特徴的な比喩・言い回しの一致
   - **レベルC(問題なし)**: 一般的な技術用語・定型表現のみの一致
4. レベルA・Bと判定した箇所には「事実は保持したまま、構成・表現を自分の言葉に書き換える」具体的な書き換え案を提示する
5. `references/manual-check-guide.md` の手動チェックをユーザーに案内する

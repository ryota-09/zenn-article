# 公開前セルフチェックリスト

記事執筆後、公開を提案する前に以下を確認する。

- [ ] AI共著記事の場合、冒頭に `:::message この記事はAIとの共著です。 :::` がある
- [ ] まとめセクションで✅絵文字を使っていない(`- **要点**: 説明` 形式になっている)
- [ ] コードブロックに言語指定とファイル名がある(` ```typescript:filename.ts `)
- [ ] 画像パスは `/images/` から始まる絶対パスになっている(相対パス不可)
- [ ] 記事末尾に「最後まで読んでいただきありがとうございます!」等の定型文がある
- [ ] title が40文字以内(推奨。Zennのハード上限は70文字)
- [ ] topics が5個以内
- [ ] 秘匿情報(ID・パスワード・トークン等)がプレースホルダー化されている
- [ ] 調査結果の文章をそのまま写しておらず、自分の言葉で書き下ろしている
- [ ] `published: false` になっている(ユーザーの公開指示があるまで)
